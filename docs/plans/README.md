# Plans Directory

This directory is for active implementation planning.

## Plan Lifecycle

1. Create a plan in `docs/plans/` while a feature is still being designed or
   implemented.
2. Keep the plan current while work is in progress.
3. After implementation, promote durable system-state outcomes to
   `docs/deep-dive/`.
4. Reduce the plan document to a concise historical pointer (status, date,
   implemented surface, deep-dive link).
